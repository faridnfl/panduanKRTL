Langkah 1: Persiapan Awal

  1. Pastikan Steam Terinstal

    Aplikasi Steam wajib terinstal dan dalam kondisi login (akun utama atau akun baru diperbolehkan).

  2. Instal .NET 8.0 Desktop Runtime

    Wajib menginstal .NET 8.0 Desktop Runtime agar launcher dapat dijalankan.

    *Jika launcher KRTL tidak terbuka/error, instalasi komponen ini belum dilakukan.

  3. Siapkan Kunci Lisensi

    Pastikan Anda sudah memiliki Kunci Lisensi (Key) yang valid untuk mengaktifkan launcher KRTL saat pertama kali dibuka.

Langkah 2: Aktivasi Launcher

  1. Instal & Buka Launcher

    Jalankan file installer KRTL_Installer.exe dan buka aplikasi KRTL.

  2. Masukkan Lisensi

    Saat pertama kali dibuka, akan muncul permintaan kunci. Masukkan Kunci Lisensi.

    *Lisensi akan terkunci otomatis pada perangkat anda.

Langkah 3: Menambahkan Game

Buka menu (‚â°) lalu pilih "Tambah Game". Anda akan melihat dua tombol angka (1 dan 2) di pojok kiri atas menu sebagai pilihan metode:

  1. Metode PIN (Manual)

    Gunakan tab ini jika mendapatkan AppID dan PIN khusus:

    • Masukkan AppID game pada kolom pertama.
    • Masukkan PIN akses pada kolom kedua.
    • Klik Tambahkan dan tunggu hingga notifikasi muncul.
  2. Metode Tiket (Otomatis)

    Gunakan tab ini untuk aktivasi instan:

    • Masukkan AppID game yang ingin dimainkan.
    • Klik Tambahkan. KRTL akan otomatis memasang data game tersebut.

Setelah muncul notifikasi sukses, game akan otomatis masuk ke daftar Library Game tanpa perlu disimpan manual lagi.

Langkah 4: Cara Bermain

‚ö†Ô∏è PERHATIKAN: Download File Game

Sebelum menjalankan game, Perlu mendownload file gamenya melalui Steam terlebih dahulu:

  1. Pilih game di KRTL, lalu klik Restart Steam.
  2. Setelah steam sudah terbuka, pergi ke Library Steam anda.
  3. Jika game belum muncul di daftar library Steam, Klik Kanan pada game tersebut di KRTL lalu pilih "Library Game". Halaman install akan terbuka otomatis di Steam.
  4. Di aplikasi Steam, klik tombol Install dan tunggu hingga selesai.

Cara 1: Bermain Lewat Steam (Original)

  1. Buka menu Library Game di KRTL.
  2. Pilih judul game yang ingin dimainkan.
  3. Klik tombol Restart Steam.
  4. Setelah Steam terbuka, klik tombol Play di dalam Steam.

üí° Info Penting:
Hanya game yang Anda pilih di KRTL sebelum menekan Restart Steam yang bisa dimnainkan. Game lain akan tetap muncul sebagai "Purchase". Jika ingin ganti game, ulangi proses pilih game dan restart steam di KRTL.

Cara 2: Bermain Langsung (Direct Play)

*Cara ini memungkinkan Anda membuka game tanpa masuk ke aplikasi Steam lagi.

  1. Klik Kanan pada judul game di daftar KRTL.
  2. Pilih menu Play -> Tentukan .exe.
  3. Cari dan pilih file utama game (file .exe di folder instalasi).
  4. Setelah terdaftar, Anda cukup Klik Kanan gamenya di library KRTL -> Play -> Play untuk membuka game secara instan.

Langkah: Menggunakan OnlineFix / Bypass

Gunakan fitur ini jika game yang dimainkan membutuhkan akses Multiplayer atau bypass keamanan tambahan (seperti Denuvo).

  1. Buka Menu OnlineFix / Bypass

    Klik tombol Menu (‚â°) di pojok kiri atas aplikasi, lalu pilih menu "OnlineFix / Bypass".

  2. Pilih Judul Game

    Ketik nama game pada kolom Cari Game, lalu klik judul game yang muncul di daftar hasil pencarian.

  3. Tentukan Folder Game

    Klik tombol Browse, lalu arahkan ke lokasi folder instalasi game tersebut (Contoh: steamapps/common/...).

  4. Proses Patching

    Klik tombol PATCH GAME. Tool akan otomatis mendownload file bypass terbaru dan memasangnya langsung ke folder game Anda.

  5. Selesai

    Tunggu hingga muncul notifikasi "Sukses". Setelah itu, Anda dapat menjalankan game secara normal melalui Steam atau menggunakan fitur Direct Play.

Fitur lainnya

Akses fitur teknis di bawah ini melalui klik kanan pada daftar Library Game.

G

Goldberg Setting

Ubah game menjadi Standalone..

Tahap Penggunaan:

  • Pilih Goldberg Setting pada menu klik kanan game.
  • Tentukan file utama (.exe) melalui tombol Browse pertama.
  • Tentukan file steam_api melalui tombol Browse kedua.
  • Klik Apply Goldberg untuk memproses emulator.
S

Steamless (Remove DRM)

Membantu memperbaiki game yang gagal terbuka.

Tahap Penggunaan:

  • Pilih Steamless pada menu klik kanan game.
  • Browse dan pilih file .exe utama yang ingin diperbaiki.
  • Klik Apply Remove DRM dan tunggu hingga selesai.
