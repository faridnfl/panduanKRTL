Langkah 1: Persiapan Awal

  1. Pastikan Steam Terinstal

    Aplikasi Steam wajib terinstal dan dalam kondisi login (akun utama atau akun baru diperbolehkan).

  2. Instal .NET 8.0 Desktop Runtime

    Wajib menginstal .NET 8.0 Desktop Runtime agar launcher dapat dijalankan.

    *Jika launcher KRTL tidak terbuka/error, instalasi komponen ini belum dilakukan.

  3. Siapkan Kunci Lisensi

    Pastikan Anda sudah memiliki Kunci Lisensi (Key) yang valid untuk mengaktifkan launcher KRTL saat pertama kali dibuka.

  4. Siapkan Kode Akses Game

    Pastikan AppID dan PIN untuk game yang ingin Anda mainkan sudah tersedia.

Langkah 2: Aktivasi Launcher

  1. Instal & Buka Launcher

    Jalankan file installer KRTL_Installer.exe dan buka aplikasi KRTL.

  2. Masukkan Lisensi

    Saat pertama kali dibuka, akan muncul permintaan kunci. Masukkan Kunci Lisensi.

    *Lisensi akan terkunci otomatis pada perangkat anda.

Langkah 3: Menambahkan Game

  1. Buka Menu Tambah Game

    Klik tombol Menu (‚â°) di pojok kiri atas aplikasi, lalu pilih menu "Tambah Game".

  2. Masukkan Kode Akses

    Isi kolom App ID dan PIN sesuai yang diterima.

  3. Klik Tombol Tambahkan

    Klik tombol "Tambahkan". Aplikasi akan otomatis mendownload konfigurasi game dari server.

    Tunggu hingga muncul notifikasi "Sukses".

  4. Simpan ke Library

    Setelah unduhan berhasil. Klik tombol "Simpan ke Library" untuk menyimpan game secara permanen.

Langkah 4: Cara Bermain (Library Game)

Buka menu (‚â°) lalu pilih "Library Game".

  1. Tentukan Folder Steam

    Pastikan lokasi instalasi Steam sudah benar pada kolom Folder Steam.

    Contoh: C:\Program Files (x86)\Steam

  2. Pilih Game & Restart Steam

    Pilih judul game yang ingin dimainkan dari daftar, lalu klik tombol "Restart Steam".

    *KRTL akan membuka Steam secara otomatis.

  3. Memunculkan Game di Library Steam

    Jika game belum muncul di daftar Library Steam atau baru mau diinstall, lakukan langkah berikut pada tool KRTL:

    • Klik Kanan pada nama game di daftar game Tool KRTL.
    • Pilih menu "Library Game".
    • Halaman detail game tersebut akan otomatis terbuka di Steam Anda.

Langkah 5: Mengubah Target Launcher (Khusus)

Lakukan tahap ini HANYA JIKA Diinstruksikan.

  1. Buka Folder Game via KRTL

    Buka tool KRTL, lalu pada daftar Library Game:

    • Klik Kanan pada judul game yang dimaksud.
    • Pilih menu "Folder Game".

    Windows Explorer akan otomatis terbuka langsung di lokasi instalasi game tersebut.

  2. Salin Jalur File Bypass

    Cari file .exe bypass yang dimaksud (Contoh: Launcher.exe atau Bypass.exe).

    Klik kanan file tersebut -> Pilih "Copy as path" (Salin sebagai jalur).

  3. Buka Properties di Steam

    Kembali ke Library Steam. Klik Kanan pada Game -> Pilih menu Properties...

  4. Atur Launch Options

    Pada tab General, cari kolom LAUNCH OPTIONS di bagian paling bawah.

    Tempelkan (Paste) jalur file yang tadi disalin, lalu tambahkan %command% di belakangnya.

    Format Wajib:

    "D:\SteamLibrary\steamapps\common\NamaGame\Bypass.exe" %command%

    ‚ö†Ô∏è PERHATIKAN:
    1. Tanda kutip "" di awal dan akhir jalur file wajib ada.
    2. Pastikan ada spasi sebelum mengetik %command%.

  5. Selesai

    Tutup jendela Properties dan klik Play di Steam. Steam sekarang akan menjalankan file bypass tersebut secara otomatis.

Fitur Tambahan (Opsional)

Menu: OnlineFix / Bypass
Gunakan menu ini HANYA JIKA game tersebut membutuhkan Crack/Bypass tambahan (umumnya untuk game Denuvo atau Multiplayer).

Caranya:
  1. Buka menu OnlineFix.
  2. Cari nama game di kolom pencarian.
  3. Pilih folder instalasi game (steamapps/common/NamaGame).
  4. Klik tombol Patch Game dan tunggu proses selesai.
Menu: Steamless
Menu ini digunakan untuk menghapus DRM SteamStub secara manual. Tidak perlu digunakan kecuali terdapat instruksi khusus.

Pusat Bantuan (FAQ)

Q: Tombol "Buka Steam" tidak bisa diklik?
Pastikan game sudah dipilih pada dropdown. Jika ini instalasi pertama, pastikan langkah "Simpan ke Library" pada menu Tambah Game telah dilakukan.
Q: Muncul tombol "PURCHASE" di Steam?
Tutup Steam sepenuhnya (Exit dari tray icon di pojok kanan bawah Windows). Buka kembali Launcher KRTL, pilih game, dan klik "Buka Steam" lagi.
Q: Gagal Download / Koneksi Error saat Tambah Game?
Gunakan DNS 1.1.1.1 (Cloudflare) atau VPN, karena kemungkinan akses ke server database terblokir oleh penyedia layanan internet.
