Langkah 1: Persiapan Awal

  1. Pastikan Steam Terinstal

    Aplikasi Steam wajib terinstal dan dalam kondisi login (akun utama atau akun baru diperbolehkan).

  2. Instal .NET 8.0 Desktop Runtime

    Wajib menginstal .NET 8.0 Desktop Runtime agar launcher dapat dijalankan.

    *Jika launcher KRTL tidak terbuka/error, instalasi komponen ini belum dilakukan.

  3. Siapkan Kunci Lisensi

    Pastikan Anda sudah memiliki Kunci Lisensi (Key) yang valid untuk mengaktifkan launcher KRTL saat pertama kali dibuka.

Langkah 2: Aktivasi Launcher

  1. Instal & Buka Launcher

    Jalankan file installer KRTL_Installer.exe dan buka aplikasi KRTL.

  2. Masukkan Lisensi

    Saat pertama kali dibuka, akan muncul permintaan kunci. Masukkan Kunci Lisensi.

    *Lisensi akan terkunci otomatis pada perangkat anda.

Langkah 4: Download Game

KRTL menyediakan dua metode untuk mendownload game.

1 Download Lewat Aplikasi Steam

Metode standar, download langsung menggunakan Steam.

  1. Buka menu Library Game di KRTL, lalu pilih game yang ingin didownload.
  2. Klik tombol Restart Steam di KRTL. Tunggu hingga Steam terbuka kembali.
  3. Buka Library di aplikasi Steam Anda.
  4. Penting: Jika tombol Install belum muncul di Steam:
    • Kembali ke KRTL, Klik Kanan pada nama game tersebut.
    • Pilih opsi "Library Game".
    Ini akan memicu halaman instalasi di Steam.
  5. Tombol Install akan muncul di Steam. Klik dan lakukan download seperti biasa.

2 Download Lewat KRTL

Metode alternatif untuk download tanpa menggunakan steam.

  1. Buka menu Library Game di KRTL.
  2. Klik Kanan pada judul game, lalu pilih "DDM".
  3. Klik Browse untuk memilih lokasi penyimpanan.
    *Tips: Disarankan membuat folder baru dengan nama yang sesuai dengan gamenya.
  4. Klik tombol START DOWNLOAD untuk memulai proses.

Langkah 5: Cara Bermain

Metode bermain tergantung dari mana Anda mendownload game tersebut (Via Steam atau Via KRTL).

A. Jika Game Didownload dari Steam

Opsi 1: Main Lewat Launcher Steam (Standar)
  1. Buka menu Library Game di KRTL -> Pilih Game.
  2. Klik tombol Restart Steam.
  3. Setelah Steam terbuka, masuk ke Library Steam.
  4. Pilih gamenya lalu klik tombol Play di Steam.
Opsi 2: Direct Play (Tanpa Buka Steam UI)

*Gunakan cara ini untuk membuka game lebih cepat lewat KRTL.

  1. Di KRTL, Klik Kanan gamenya -> Pilih menu Play -> Tentukan .exe.
  2. Cari file .exe game di folder instalasi Steam.
    (Biasanya di: Steam\steamapps\common\NamaGame).
  3. Setelah ditentukan, mainkan dengan cara: Klik Kanan -> Play -> Play.

B. Jika Game Didownload dari KRTL

Karena game didownload terpisah (Portable/Standalone), Anda WAJIB mengatur Emulator (Goldberg) agar game bisa jalan tanpa Steam.

  1. Tentukan File .exe
    • Klik Kanan gamenya -> Play -> Tentukan .exe.
    • Arahkan ke folder tempat Anda mendownload game tadi via KRTL.
  2. Pasang Emulator (WAJIB)
    • Klik Kanan gamenya -> Pilih Goldberg Setting.
    • Isi kolom EXE Game (file yang tadi dipilih).
    • Isi kolom Steam API (Cari file steam_api.dll atau steam_api64.dll di folder game).
    • Klik tombol Apply Goldberg dan tunggu notifikasi sukses.
  3. Mainkan Game
    • Klik Kanan gamenya -> Play -> Play.

Langkah: Menggunakan OnlineFix / Bypass

Gunakan fitur ini jika game yang dimainkan membutuhkan akses Multiplayer atau bypass keamanan tambahan.

  1. Buka Menu OnlineFix / Bypass

    Klik tombol Menu (‚â°) di pojok kiri atas aplikasi, lalu pilih menu "OnlineFix / Bypass".

  2. Pilih Judul Game

    Ketik nama game pada kolom Cari Game, lalu klik judul game yang muncul di daftar hasil pencarian.

  3. Tentukan Folder Game

    Klik tombol Browse, lalu arahkan ke lokasi folder instalasi game tersebut.

  4. Proses Patching

    Klik tombol PATCH GAME. Tool akan otomatis mendownload file bypass terbaru dan memasangnya langsung ke folder game Anda.

Fitur lainnya

Akses fitur teknis di bawah ini melalui klik kanan pada daftar Library Game.

G

Goldberg Setting

Ubah game menjadi Standalone (Offline tanpa Steam).

  • Pilih Goldberg Setting pada menu klik kanan game.
  • Tentukan file utama (.exe) dan steam_api.
  • Klik Apply Goldberg.
S

Steamless (Remove DRM)

Membantu memperbaiki game yang gagal terbuka karena proteksi Steam Stub.

  • Pilih Steamless pada menu klik kanan game.
  • Browse dan pilih file .exe utama yang ingin diperbaiki.
  • Klik Apply Remove DRM.
